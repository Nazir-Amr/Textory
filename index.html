Overview

What is Textory?

Textory is a lightweight desktop application built with Python. It gives you a clean, organized space to store and manage reusable text snippets — responses, code notes, templates, and more — without any cloud service, account, or internet connection required.

🐍

Built with Python

Lightweight and native. Runs directly on your desktop without background services or heavy runtimes.

💾

100% Local Storage

Your data never leaves your device. No accounts, no sync, no cloud — just your files, on your machine.

⚡

Built for Speed

Copy, edit, and search in seconds. Optimized for professionals who deal with repetitive text every day.

🌐

Multilingual

English, German, and Arabic — with proper right-to-left layout support for Arabic users.

Key Features

Everything You Need

A focused, no-bloat feature set designed for people who work with text every single day.

📝 Text Card Management

Each snippet is stored as a card. You can save, edit, copy to clipboard, and delete cards with a single click. Cards display a title, a preview of the content, and quick-action buttons — making it fast to find and reuse what you need.

🔍 Live Search

Start typing and results appear instantly. The search filters across all stored titles and content in real time, so there's no need to scroll through a long list to find a specific snippet.

📂 Groups & Categories

Organize your snippets into custom groups. Assign any card to a group and filter by group to quickly narrow down the list — ideal when you have dozens of snippets across different topics or projects.

👁 Right-Side Preview Panel

Click a card to load its full content in the right-side panel without opening the editor. This lets you read the complete text and decide whether to copy or edit — without any extra windows in the way.

✏️ Rich Text Formatting

  • Bold, Italic, Underline, Strikethrough — standard inline formatting
  • Text Highlighting — mark key phrases for visual emphasis

All formatting is preserved when copying and round-trips correctly on edit — what you write is what you paste.

📤 Import / Export System

  • Export selected snippets or your entire library
  • Import from a file with a built-in selection dialog
  • Search within titles during import to find specific entries
  • Duplicate detection — Textory warns you and lets you choose: skip or replace

🌍 Multi-Language Support

  • English, German (Deutsch), Arabic (العربية)
  • Full right-to-left (RTL) layout for Arabic
  • Switch language from the Settings without restarting

🌗 Light & Dark Mode

Switch between a clean light interface and a comfortable dark theme. The preference is saved automatically and applied on the next launch.

👤 Username Attribution

Every card records who created and who last edited it. When working in shared environments or logging your own edits over time, you always know who changed what.

📖 Built-In Instructions

Textory includes an in-app instructions page. New users can learn how to use all features without searching for external documentation — everything is explained inside the application itself.

Interface

How the Interface Works

Textory follows a clear, predictable layout. Every window has a single purpose — no hidden menus, no cluttered toolbars.

01 · MAIN WINDOW

Your Snippet Library

The main window shows all your cards in a scrollable list on the left. The right side displays a full-content preview of the selected card. A search bar sits at the top, and group filters let you narrow the view instantly. Buttons for adding, editing, and deleting are always visible and clearly labeled.

02 · ADD / EDIT WINDOW

The Editor

A focused editing window with a title field, a group selector, and a rich text editor. The formatting toolbar gives you Bold, Italic, Underline, Strikethrough, and Highlight controls. Once saved, the card appears immediately in the main list.

03 · SETTINGS

Preferences Panel

A clean settings panel lets you change the display language, toggle between light and dark mode, and set your username. Changes apply instantly — no restart required for most settings.

04 · IMPORT / EXPORT

Data Dialogs

The import dialog shows all available entries from a file. You can search within them, select individual items, and choose how to handle duplicates. Export works the same way — select what you want to share or back up, and save to a file.

Textory is built for anyone whose daily work involves repeating the same text over and over. If you find yourself copying from a previous chat, an old document, or a sticky note just to paste it somewhere else — Textory is for you.

Keep code snippets, terminal commands, boilerplate blocks, and configuration templates always at hand.

🤝

Customer Service
